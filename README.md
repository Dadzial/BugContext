# BugContext – Smart Log Analyzer & Bug Reporter

BugContext is a lightweight web application that helps QA testers analyze system logs and generate bug reports instantly using the Gemini API.

##  Key Feature
* **AI Log Analysis:** Translates complex server logs and stack traces into clear, human-readable explanations.


## 🛠 Tech Stack
* **Backend:** Python 3, Flask
* **AI:** Gemini API (`google-genai`)
* **Frontend:** HTML5, CSS, JavaScript
